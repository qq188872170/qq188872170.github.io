# 常见问题

# 💡登陆了账户，但不出现VIP标志？

解决方案

  1. 确保自己是在文档下载页面，首页不会出现VIP标志

  2. 尝试刷新几次网页

  3. 上述无果，检查是否安装有其他文库脚本，大多数文库脚本会导致VIP功能破解失败

    • 点击浏览器右上角油猴标志，然后会弹出当前已匹配的油猴脚本
    • 关闭其他所有百度文库脚本，只开启我们的油猴脚本，如下图

  4. 如果还是不出现VIP标志，更换其他文档，或者联系作者，让其尝试是否能下载

# 💡不出现下载按钮？

解决方案

先尝试使用Edge浏览器或者谷歌浏览器，如果还出现这种情况，请尝试下面的方法

键盘按 F12键 或者在文档正文以外的屏幕区域点击鼠标右键选择 审查元素

然后点击左下角这个按钮，切换成 手机模式，然后刷新网页。

然后就出现下载按钮了，然后点击下载按钮，输入验证码，点击下载PDF

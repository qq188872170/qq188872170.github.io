# 如何离线安装油猴扩展？

Chrome/Edge 扩展商店经常无法访问或速度很慢，使得在线安装扩展成为奢侈，好在各浏览器均支持离线安装扩展，安装效果和在线安装完全一样。

首先需要知道浏览器扩展后缀为 .crx 或 .xpi。

#
Chrome浏览器

点击查看

在 crxsoso.com (opens new window) 在线安装扩展时会提示：无法从该网站添加扩展程序 ，这是因为 Google 限制，导致无法从第三方网站在线安装扩展。

可按照以下步骤离线安装扩展：

  1. 在地址栏中输入 chrome://extensions 复制链接，打开 扩展管理页面，然后再打开右上角的 【开发者模式】，然后刷新本页。

  1. 在下载内容（快捷键 Ctrl+J ）中找到刚才下载好的扩展。

    💡 提示

    若下载列表中没有找到扩展文件可点击灰色的小按钮下载离线 .crx 文件。

  2. 将下载好的 crx 扩展文件拖动到扩展管理页面。

  1. 松开鼠标，浏览器弹出提示要添加"xxx"吗？，点击【添加扩展】，安装完成。

如果安装成功，恭喜您，请忽略以下步骤。

如果安装失败，出现以下几种情况：

  • 程序包无效：CRX_REQUIRED_PROOF_MISSING

  • 该扩展程序未列在 Chrome 网上应用店中，并可能是在您不知情的情况下添加的

  • 扩展的按钮开关为灰色，无法启用

此时将文件后缀名 crx 改为 zip，把 zip 文件拖动到扩展管理页面。（若拖动没反应，请刷新本页）

若依然无效，则解压 zip 文件，回到浏览器扩展页面，点击【加载解压缩的扩展】按钮，选择刚才解压的文件夹，扩展安装成功。

#
Edge浏览器

点击查看

在 crxsoso.com (opens new window)下载的 crx 扩展安装时会提示：xxx.crx 可能会损害你的设备。是否仍要保留？，此时选择 保留 => 另存为，将 crx 文件存放到电脑中。

若提示 无法从该网站添加扩展程序，这是因为 Edge 限制，导致无法从第三方网站在线安装扩展。

可按照以下步骤离线安装扩展：

  1. 在地址栏中输入 edge://extensions，打开扩展管理页面，然后再打开左下角的 **【开发人员模式】**和 【允许来自其他应用商店的扩展】，然后刷新本页。

    💡 提示

    打开 **【允许来自其他应用商店的扩展】**开关后，Edge 浏览器还可以安装来自 Chrome 商店中的扩展。

  2. 将下载好的 crx 扩展文件拖动到扩展管理页面。

  3. 松开鼠标，浏览器弹出提示将 "xxx" 添加到 Mircosoft Edge？，点击【添加扩展程序】，安装完成。

如果安装成功，恭喜您，请忽略以下步骤。

如果安装失败，出现以下几种情况：

  • 程序包无效：CRX_REQUIRED_PROOF_MISSING

  • 该扩展程序未列在 Edge 网上应用店中，并可能是在您不知情的情况下添加的

  • 扩展的按钮开关为灰色，无法启用

此时将文件后缀名 crx 改为 zip，把 zip 文件拖动到扩展管理页面。（若拖动没反应，请刷新本页）

若依然无效，则解压 zip 文件，回到浏览器扩展页面，点击【加载解压缩的扩展】按钮，选择刚才解压的文件夹，扩展安装成功。
